## realize

ホットリローダー

`go get -u github.com/oxequa/realize`

### usage
1. initialize
```
% realize init
```
- `.realize.yaml` が作成される


1. run
```
% realize start [--server [--open]]```
```
- `--server` オプションをつけると対象プロジェクトの状況を確認できるようになる
  - http://localhost:5002
- `--open` オプションをつけるとブラウザが開く
